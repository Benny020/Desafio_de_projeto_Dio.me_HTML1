O Que São Tags No Html?

Os elementos Html ou também chamados de tags Html, que traduzindo do inglês, Tag significa etiqueta ou rótulo.
As tags são etiquetas que não aparecem na tela do computador, mas que servem para orientar alguém.

São utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.
Com algumas contendo Tag de abertura e fechamento e outras não.



O Que São Atributos De Uma Tag?

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML.
Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc.

Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).



Tags De Texto

As tags de texto são responsáveis pela formatação do mesmo. São utilizados para organizar, sinalizar e modificar a estrutura de textos, deixando-os harmoniosos.

Tags <i>, <u>, <strong>

  • Tag < i >

    Utilizado para deixar palavras em Itálico.

  • Tag < u >

    Utilizado para deixar palavras Sublinhadas.

  • Tag < strong >

    Utilizado para deixar palavras em Negrito.



Tag < mark >

Utilizado para destacar palavras, como uma espécie de marcador digital.
